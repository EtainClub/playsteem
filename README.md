# Play Steem

PLAY STEEM Mobile App Repository

# Run on Android

```
 $ yarn
 $ yarn start
 $ react-native run android
```

# Run on ios

```
$ yarn
$ cd ios
$ pod install
$ cd ..
$ react-native run-ios
```
